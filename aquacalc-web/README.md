# AquaCalc Web — Калькулятор параметров воды для аквариума

Веб-версия калькулятора минерализации воды для разведения аквариумных рыб (L-сомы и др.).

## 🚀 Развёртывание

### GitHub Pages
1. Создайте репозиторий на GitHub
2. Загрузите файлы (`index.html`, `style.css`, `script.js`)
3. В настройках репозитория → Pages → выберите ветку `main`
4. Сайт будет доступен по адресу `https://ваш-username.github.io/имя-репозитория`

### Netlify
1. Зарегистрируйтесь на [netlify.com](https://netlify.com)
2. Перетащите папку с файлами в окно браузера
3. Сайт опубликован!

### Cloudflare Pages
1. Зарегистрируйтесь на [pages.cloudflare.com](https://pages.cloudflare.com)
2. Подключите GitHub-репозиторий или загрузите файлы напрямую
3. Развёртывание автоматическое

## 📋 Функционал

- ✅ Расчёт дозировок солей (Ca, Mg, Na, K)
- ✅ Учёт ионного состава воды
- ✅ Расчёт TDS и жёсткости в CaCO₃
- ✅ Сохранение настроек в браузере (localStorage)
- ✅ Экспорт отчёта в HTML
- ✅ Адаптивный дизайн (мобильные + десктоп)
- ✅ Справочная информация по параметрам

## 🔧 Локальный запуск

Просто откройте `index.html` в любом браузере. Сервер не требуется.
